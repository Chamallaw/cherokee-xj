# 🔥 Contacteur Comfort/Power

## Contacteur Comfort/Power

Le commutateur **Comfort/Power** se situe au tableau de bord.\
Il modifie la logique de passage des rapports selon deux cartographies prédéfinies.

{% hint style="info" %}
Ce sélecteur change le comportement de la boîte automatique AW4.
{% endhint %}

<div align="left" data-full-width="false"><figure><img src="../../../.gitbook/assets/image.png" alt="" width="188"><figcaption></figcaption></figure></div>

### À quoi sert ce commutateur

Le mode choisi agit sur le régime moteur auquel la boîte passe le rapport supérieur.\
Il agit aussi sur la rapidité du rétrogradage à l’accélération.

En pratique :

* **Power** privilégie la performance
* **Comfort** privilégie la souplesse
* le comportement change nettement à forte charge

### Les deux modes

{% columns %}
{% column %}
#### Position Power

Ce mode rend la boîte plus réactive.\
Les rapports montent plus haut dans les tours.

Sous forte accélération, le rétrogradage intervient plus tôt.\
Le moteur reste donc plus souvent dans sa plage utile.

Pour une conduite plus dynamique, c’est le mode à utiliser.
{% endcolumn %}

{% column %}
#### Position Comfort

Ce mode convient à une conduite normale.\
La boîte passe les rapports plus tôt.

Le régime moteur reste plus bas.\
Cela favorise le confort et réduit la consommation de carburant.

Pour un usage quotidien, c’est le réglage le plus adapté.
{% endcolumn %}
{% endcolumns %}

### Différence à pleine accélération

Quand l’accélérateur est enfoncé à fond, l’écart entre les deux modes devient très visible :

* en **Power**, les rapports supérieurs passent vers **5000 tr/min**
* en **Comfort**, ils passent vers **4500 tr/min**

Cette différence explique le caractère plus sportif du mode **Power**.

{% hint style="success" %}
En résumé, utilisez **Power** pour une conduite sportive.\
Utilisez **Comfort** pour une conduite souple et économique.
{% endhint %}
